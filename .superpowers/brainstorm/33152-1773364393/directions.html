C: "Minimal Dark"

Dark, clean, developer-oriented. Green accent from CriticMarkup insertions as the brand color. Tab navigation between the three pages. Quick-start command block. No decoration beyond what communicates.

What's the same across all three

  • Pure .md files — content lives in markdown, client-side JS renders it
  • Three pages — index, spec, ideas (each a separate .md file)
  • CriticMarkup colors — green (#3fb950) for insertions, red (#f85149) for deletions
  • GitHub Pages — static hosting, DNS points ctracks.com there
  • No build step — edit .md, push, done
